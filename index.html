О AnonimGram

AnonimGram — это анонимный мессенджер, созданный для тех, кто ценит свою приватность и хочет общаться без привязки к реальному миру.

Мы не собираем ваши данные, не отслеживаем вашу активность и не продаем вашу информацию. Вы — хозяин своих данных, мы просто предоставляем инструмент для свободного общения.

Основные особенности:

  • Полная анонимность — без телефона, email или социальных сетей
  • Локальное хранение данных — ваши сообщения всегда с вами
  • Минимальный сервер — только необходимые данные для работы чатов
  • Без рекламы и слежки

Проект находится в активной разработке. Следите за новостями!

4. Что хранит сервер AnonimGram
═══════════════════════════════════════════════════════════════
AnonimGram использует минимальный сервер для работы онлайн-чатов.
Он НЕ собирает и НЕ хранит:
IP-адреса, MAC-адреса, геолокацию, контакты;
Содержимое сообщений, медиафайлы, голосовые;
Временные метки активности (кроме последнего входа самого пользователя);
Любые данные, привязанные к реальному миру.
Сервер временно хранит ТОЛЬКО следующие данные, необходимые для работы интерфейса:
👤 Для пользователей:
Псевдоним (логин) — выбран при первом запуске (например: user123);
Анонимный ID — внутренний номер, присвоенный сервером;
Путь к аватару — файл изображения, загруженный пользователем;
Время последнего входа — только для статуса «был в сети» (можно скрыть);
Список чатов — только ID групп и каналов, в которых состоит пользователь.
💬 Для групп и каналов:
Название чата — задаётся создателем;
Тип — group или channel;
Уникальный ID — длинный числовой идентификатор (10 цифр для групп, 18 для каналов), сгенерированный клиентом;
ID владельца и список участников — только анонимные ID, без имён;
Последнее сообщение и время его отправки — только для отображения превью в списке чатов.
Важно:
Группы и каналы НЕ являются публичными. Их можно найти ТОЛЬКО по точному ID;
Сервер НЕ хранит историю сообщений — только одно последнее для интерфейса;
Все данные хранятся ТОЛЬКО в оперативной памяти и полностью исчезают при перезапуске сервера;
Даже при компрометации сервера злоумышленник не получит содержимого переписок или реальных имён.
Логи событий (например: «пользователь создал канал») автоматически удаляются каждые 24 часа ровно.

5. Оффлайн-режим: ваши данные — всегда с вами
═══════════════════════════════════════════════════════════════
AnonimGram полностью загружает чаты, каналы, историю переписок,
аватары и настройки даже без интернета — потому что всё хранится
локально на вашем устройстве.
Однако новые сообщения нельзя отправить в оффлайне,
и они не будут отправлены автоматически при восстановлении связи.
Это сделано намеренно:
Чтобы предотвратить перегрузку сервера;
Чтобы сохранить прозрачность: вы всегда точно знаете,
доставлено ли сообщение или нет.
Объём локальных данных оптимизирован: даже при активном использовании
он обычно не превышает 50 МБ.═══════════════════════════════════════════════════════════════
